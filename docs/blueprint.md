# **App Name**: PixelPro AI

## Core Features:

- Background Remover: Remove backgrounds from images using PixelPro AI's API. Display original and background-removed images side-by-side.
- Image Compressor: Compress images using PixelPro AI's API with a quality slider to adjust the level of compression. Display file size savings.
- AI Upscaler: Upscale images using PixelPro AI's API to enhance detail. Provide a side-by-side zoom view for comparison.
- Manual Enhancer: Manually enhance images using PixelPro AI's API with sliders for brightness, contrast, saturation, and sharpness. Implement debouncing for API calls.
- Image Resizer: Resize images by specifying width and height. Allow users to maintain aspect ratio. Send image resizing parameters to PixelPro AI tool via the api
- Background Changer: Change image backgrounds by uploading a new background image or selecting a background color. Use PixelPro AI as a tool for background change processing
- API Proxy: Securely proxy requests to the PixelPro AI API, managing the API key on the server-side to prevent client-side exposure.

## Style Guidelines:

- Primary color: Deep Purple/Indigo (#4F46E5) to evoke trust and modernity. Used for primary interactive elements such as buttons.
- Secondary color: Light Lilac/Lavender (#F0EEFF) to create soft, spacious backgrounds for sections. A lighter tint of indigo to imply a natural affinity.
- Neutral colors: White (#FFFFFF) and slate grays for text. Using darker shades of gray for text can create more depth to the overall design
- Font: 'Inter', a sans-serif font, to apply a clean, modern look globally.
- Use Lucide React icons for a consistent and clean design.
- Implement a sticky glassmorphism header for improved navigation.
- Apply subtle fade-in effects, hover effects, and smooth transitions using Framer Motion for a premium user experience.
